# Apple Music BG Loop

Небольшой генератор анимированного фона под картинку, вдохновленный фоном в полноэкранном режиме Apple Music

## Что делает программа

1. Принимает картинку через `input[type=file]`
2. Сжимает ее до маленького буфера и считает цветовые кластеры
3. Для каждого кластера считает:
- долю цвета в изображении
- центр массы цвета в кадре
- концентрацию цвета в пространстве
- общую резкость переходов
4. Передает это в шейдер
5. Шейдер строит аморфные пятна с учетом:
- весов цветов
- локальных центров цветов
- концентрации и резкости
6. При записи пишет ровно 10 секунд в `webm vp9`, чтобы луп замыкался

## Режимы UI

- `движение`
- `soft` спокойнее
- `balanced` средний режим
- `active` более выраженная динамика
- `текстура`
- `clean` почти без зерна
- `balanced` умеренно
- `grain` более заметная пленочность

## Стек

- `TypeScript`
- `Vite`
- `WebGL 1`
- `MediaRecorder`

## Структура

- `index.html` - UI и точка входа
- `src/main.ts` - события UI и склейка модулей
- `src/palette.ts` - анализ картинки
- `src/renderer.ts` - WebGL-рендер и uniform-состояние
- `src/shaders.ts` - GLSL-код шейдеров
- `src/recorder.ts` - экспорт 10s loop в `webm`
- `src/style.css` - стили интерфейса
- `src/types.ts` - общие типы
- `vite.config.ts` - конфиг dev/build
- `tsconfig.json` - конфиг TypeScript
